Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models

🔴 Tencent Hunyuan📄 arXiv▲ 10
2606.11324 · arXiv · code

사용자 instruction과 image observation을 입력받아 spatial grounding(pointing)·trajectory·natural language reasoning을 출력하는 8B EFM. cognition·planning·pointing을 단일 모델로 통합해 long-horizon manipulation을 self-correction 포함 closed-loop으로 자율 수행. Tianjin Univ + Tencent Hunyuan.

기여pointing specialist였던 선행 Embodied-R1을 cognition·planning·pointing 통합 종합 EFM으로 확장하고, multi-task balanced RL recipe + Planner-Grounder-Corrector(PGC) closed-loop로 자율적 long-horizon 실행과 failure correction을 달성.
강점
  • 24개 embodied VLM 벤치 중 16개 SOTA — 주요 21개 평균 70.4%로 Gemini-Robotics-ER-1.5 대비 +17.0%
  • LIBERO에서 action pretraining 없이 97.3% — 대규모 action data 기반 방법과 동급
  • 3개 자동화 데이터 파이프라인으로 15B+ token 데이터 시스템 구축, 모델·데이터·EvalKit 전부 공개
  • 실로봇 zero-shot에서 instruction following·affordance grounding·articulated manipulation 검증
약점
  • 복잡한 3D manipulation 실로봇 성공률 낮음 (Door Open 16.6%, Cup Disassembly 66.6%)
  • PGC memory가 단순 FIFO buffer — 저자도 더 정교한 memory 전략 필요성 인정
  • Planner-Grounder-Corrector 다중 inference call 조율로 배포 시 추론 비용·복잡도 증가
  • cognition/spatial reasoning에서의 우위 margin은 pointing 대비 상대적으로 작음

DAM-VLA: Decoupled Asynchronous Multimodal Vision Language Action model

🔴 NVIDIA📄 arXiv
2606.12105 · arXiv · code

RGB(25Hz)·proprioception(100Hz)·force/torque(100Hz)·language를 modality별 latent buffer에 각자 센서 주기로 비동기 갱신하며 manipulation action을 생성하는 VLA. contact-rich 7개 실로봇 태스크 평균 95.2%를 100Hz smooth·reactive 제어로 달성. KIT 주도 + NVIDIA.

기여모든 센서를 단일 clock에 동기화하던 기존 VLA와 달리 modality별 비동기 latent buffer(decoupled temporal processing)를 제안하고, gated cross-attention으로 고주파 modality를 pretrained VLM backbone 손상 없이 통합.
강점
  • 7개 contact-rich 태스크 평균 95.2% vs 최강 동기 baseline 40.95% (+54.25%p)
  • 동기 baseline이 거의 0%인 socket insertion 80.0%, Lego arranging 93.3%
  • 100Hz 설정 중 최저 SPARC(8.1)로 가장 smooth한 action, tracking lag 0.116s (25Hz baseline 0.189s)
  • RTX 4060 Ti 단일 GPU에서 100Hz 반응 제어
약점
  • action chunk 내부에서 force 신호로 action을 보정하지 않아 극단적 contact-heavy 태스크 제한 (socket insertion 80%)
  • 단일 플랫폼(Franka Panda) 7개 태스크 평가 — 일반화 검증 부족
  • vision decoupling이 불완전 (scene 변화가 아닌 고정 timer 기반 camera update)
  • 전용 F/T 센서 없이 Franka 내장 joint-torque 추정 의존 — 타 플랫폼 확장성 미검증

Real-Time Execution with Autoregressive Policies

🔴 Google Research📄 arXiv
2606.13355 · arXiv · code

Autoregressive VLA(π₀-FAST 계열)를 비동기 inference로 실시간 실행: tokenization horizon 조정으로 action horizon과 inference 주기를 분리하고, constrained decoding(coin-change DP logit masking)으로 latency 상한을 보장하며, multi-trajectory decoding으로 성능을 보강 (π₀-REALFAST). KIST+SNU+Google Research.

기여실시간 로봇 제어가 diffusion/flow-matching policy의 전유물이 아님을 입증 — constrained decoding 기반 latency 보장과 horizon-주기 분리로 AR policy의 경쟁력 있는 real-time execution 달성.
강점
  • LIBERO 평균 95.7%로 RTC 적용 π₀(89.4~94.8%)를 일관되게 상회
  • ~6k training steps(~6 epochs)만에 최고 성능 도달하는 빠른 수렴
  • DROID zero-shot 실로봇에서 π0.5에 비견, 동기식 π₀-FAST 대비 큰 폭 개선
  • 외부 latency 변동에도 동일 inference 빈도에서 강건 (RTC는 빈도 증가 시 성능 저하)
약점
  • 정적 환경의 single-arm tabletop manipulation에 평가 한정
  • bimanual/동적 환경 확장 시 더 많은 토큰 디코딩이 필요해 latency 제약 심화
  • LIBERO는 실제 실행 지연을 반영하지 못해 인위적 제약으로 real-time을 모사
  • 복잡한 시나리오용 AR action expert 아키텍처는 open question

강점

  • 트랙 점수 equal-weight 집계의 근거 부재

Monte Carlo Pass Search: 3D Counterfactual Pass Evaluation in Football

🎓 CVPR
2606.11120 · arXiv · code

축구 pass 평가를 MCTS 유사 문제로 재정의: CEM policy로 kick parameter를 추론해 local(실행 노이즈) 256 + global(대안 옵션) 256 counterfactual pass를 샘플링, SMART 기반 world model로 3D 궤적 rollout 후 Transformer value model(Possession Value)로 ΔPV 분포를 산출. 로보틱스 WM/VLA 아님(도메인 밖).

기여ball trajectory 조건부 world-model rollout 기반 pass 평가 프레임워크와 실행 민감도·기회집합을 함께 근사하는 dual Monte Carlo search, 분포 인지형 평가 지표 2종 + 코칭 시각화 제안.
강점
  • 점추정이 아닌 value distribution 기반 평가 — local/global dual sampling으로 실행 노이즈와 대안 옵션 분리 정량화
  • 3D ball trajectory 명시 조건화 rollout: minADE20 2.4 / minFDE20 4.7로 baseline 대비 우수
  • 케이스 스터디·passer ranking 등 코칭 워크플로우 직결 출력
약점
  • 학습 데이터가 Bundesliga 7경기뿐 — CV tracking 오류 상속
  • off-ball 행동·전술 관례 미반영 world model misspecification, sampler가 단순
