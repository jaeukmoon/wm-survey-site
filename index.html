WALA: Learning Executable Latent Actions from Action-Labeled Demonstrations and Action-Free Videos

필독 B 76⚔완성도 75⚡ vs FastWAM📄 arXiv
2607.11397 · arXiv

WALA는 행동 라벨이 있는 로봇 시연과 라벨이 없는 대규모 비디오를 함께 써서 '실행 가능한 잠재 행동(latent action)'을 학습하는 프레임워크다. 픽셀을 복원하는 대신 DINOv3 특징 공간과 dense depth 공간에서 현재→미래 관측의 delta를 예측해, 외형 변화에 둔감하면서 과제 관련 의미·기하 구조를 보존하는 잠재 행동 모델을 사전학습한다. 정책 학습 시 이 인코더가 안정적인 잠재 행동 타깃을, 디코더가 학습 가능한 잠재 world model 역할을 하여 라벨 있는 시연은 제어 감독을, 라벨 없는 비디오는 dynamics 감독을 제공한다.

기여행동 라벨 유무가 섞인 데이터를 하나의 잠재 행동 표현으로 통합해, 값비싼 로봇 행동 라벨 없이도 대규모 action-free 비디오에서 제어에 유효한 dynamics 감독을 끌어내는 학습 틀을 제시한다.
강점
  • 픽셀 복원 대신 DINOv3 feature + dense depth의 미래 delta를 예측 → 의미·기하 보존, 외형 민감도 감소
  • RoboCasa-GR1-Tabletop 평균 성공률 75.2%로 SOTA(차순위 DIAL 70.2% 대비 +5.0%p)
  • action-free 비디오가 dynamics 감독을 제공해 로봇 행동 주석 없이 확장 가능, 실세계 조작 일반화 개선
약점
  • RoboTwin 2.0 Clean에서는 90.6%로 LingBot-VA(92.9%)·Fast-WAM(91.9%)에 다소 뒤짐(Random에서는 역전)
  • 잠재 행동 품질이 DINOv3·depth teacher 표현에 종속 — teacher 편향이 그대로 전파될 위험
  • 시뮬(RoboTwin/RoboCasa) 위주 평가, 실세계는 소수 태스크(pick-place·stack·insert 등)로 제한

Lumo-2: Towards Predictive, Aligned, and Scalable Robot Learning

필독 C 60완성도 76⚡ vs FastWAM📄 arXiv
2607.11270 · arXiv

Lumo-2는 잠재 공간에서 world dynamics를 추론해 행동을 생성하는 잠재 world-action model이다. 학습된 잠재 world dynamics가 물리적으로 근거 있는 시각 전이를 포착해 미래 가능성을 인코딩하고, 이를 cross-modal 정렬의 공통 기반으로 삼는다. 핵심 가설은 '행동 생성 품질이 잠재 공간의 기하에 좌우된다'는 것으로, 복원 기반 tokenization이 저수준 신호 충실도에 치우쳐 제어 성능과 어긋나는 문제를 다단계 modality 사전정렬(world dynamics→vision→language)로 해소한다.

기여복원 충실도와 제어 성능의 불일치를 지적하고, 행동 표현을 잠재 world dynamics·비전·언어에 단계적으로 정렬하는 multi-stage pre-alignment로 예측적 추론에 적합한 구조화된 잠재 공간을 유도한다.
강점
  • 잠재 world dynamics 기반 예측적 추론을 경량으로 유지하면서 VLA·WAM baseline을 일관되게 능가
  • 임베디드 벤치서 4B VLM이 Qwen-3.5 4B 대비 큰 격차(VSIBench 52.34 vs 21.73, MindCube 66.08 vs 39.95)
  • long-horizon·dexterous 등 시간적 추론·물리 이해가 필요한 실세계 22개 태스크에서 이득 + scaling·OOD 일반화 체계적 분석
약점
  • 실세계 22개 태스크 성적이 본문 표가 아닌 그림(Fig.19-20)에 있어 정확한 수치 비교가 어려움
  • 핵심 주장(잠재 공간 기하가 행동 품질 좌우)이 상관 분석 위주 — 인과적 검증은 제한적
  • 다단계 정렬·대규모 코퍼스(FineVision 24.3M 등)에 의존해 재현 비용이 큼

InternVLA-A1.5: Unifying Understanding, Latent Foresight, and Action for Compositional Generalization

필독 A 80완성도 84⚡ vs FastWAM🔴 Shanghai AI Laboratory📄 arXiv▲ 24
2607.04988 · arXiv · code

이미지·언어 지시를 입력받아 연속 로봇 매니퓰레이션 액션을 출력하는 통합 정책. VLM 백본(Qwen-3.5 2B)은 VQA·서브태스크 예측 학습을 계속 유지하고, 여기에 연속 액션 생성을 위한 경량 unified expert를 붙였다. 학습 시 소수의 learnable foresight 토큰이 frozen 비디오 생성 모델(WAN2.2-5B)의 감독 아래 태스크 관련 미래를 latent code로 압축해 픽셀 생성 학습 없이 world-model 동역학 prior를 상속하며(비디오 브랜치는 추론 시 폐기해 실시간 제어 유지), 6개 시뮬레이션 벤치마크 전반 최고 성적과 실제 로봇 held-out 지시 조합에서 최강 compositional generalization을 달성.

기여기존 통합 모델이 픽셀 공간에서 미래 예측을 처음부터 학습하며 사전학습 VLM의 semantic을 훼손하고 이질적 목적 간 간섭을 겪던 문제를, 미래 예측을 'latent-querying' 문제로 재정의해 해결한 점이 새롭다. frozen 사전학습 비디오 생성기의 동역학 prior를 학습된 foresight 토큰으로 증류받아 픽셀 수준 생성을 전혀 학습하지 않고도 world-model prior를 얻으면서, 네이티브 VLM 백본의 VQA·서브태스크 학습을 유지해 semantic을 보존한다.
강점
  • 6개 시뮬레이션 벤치마크 전반 최고: SimplerEnv-WidowX 80.8%(π0.5 57.1%, GR00T-N1.5 61.9%), RoboTwin 2.0 93.2%(π0.5 79.8%), LIBERO 98.9% 등 다양한 셋업에서 일관된 우위.
  • 실제 로봇 long-horizon MOF(화학 절차) 76.4% vs π0.5 29.3% — semantic 보존과 latent foresight 두 설계가 held-out 지시 조합·장기 실행에서 압도적 격차를 만들어 compositional generalization을 실증.
  • world-model 동역학 prior를 얻으면서도 비디오 생성 브랜치를 추론 시 완전히 폐기 → 추가 추론 비용 없이 실시간 제어 유지(설계-비용 트레이드오프가 유리).
  • 대규모 사전학습(1.2M 로봇 에피소드/861M 프레임 + 3M 멀티모달 샘플)으로 스케일 근거 확보.
약점
  • 저자 인정 한계: foresight 감독이 단 1개 action chunk의 단기 horizon만 커버 → local dynamics prior만 흡수, 장기 imagination이나 world model을 이용한 명시적 planning은 아직 불가.
  • 비디오 생성기가 frozen·generic이라 상속되는 prior가 그 사전학습이 embodied scene을 얼마나 커버하느냐에 상한이 걸림(도메인 갭 시 이득 제한).
  • 일부 벤치마크는 baseline 대비 마진이 미미해 '최고'라도 실효 격차가 작음: LIBERO-Plus zero-shot 84.8 vs π0.5 84.4, DOMINO zero-shot 27.7 vs 26.6 — 통계적 유의성/재현성 불확실.
  • 절대 성공률이 낮은 어려운 영역 잔존(EBench 35.2% SR, DOMINO 27.7% SR)으로, 난이도 높은 태스크에서는 여전히 취약.

GigaWorld-1: A Roadmap to Build World Models for Robot Policy Evaluation

필독 C 60완성도 79⚡ vs FastWAM📄 arXiv▲ 37
2607.02642 · arXiv · code

로봇 정책을 느리고 비싼 실제 롤아웃 없이 평가하기 위한 액션 조건부 비디오 월드모델. 초기 프레임과 액션 인코딩(헤드 카메라용 EE-pose map, 손목 카메라용 ray map)을 입력받아 정책 롤아웃 비디오를 생성하고 그 성공/실패를 대리 판정한다. 함께 제안한 벤치마크 WMBench(실로봇 텔레오퍼레이션 + 매칭 정책 롤아웃, 8개 태스크군, 32.4만+ 시뮬 롤아웃)에서 GigaWorld-1-Plus(5B)가 종합 0.6834로 최고 베이스라인 대비 +14.9%를 달성했다.

기여기존 연구가 "월드모델을 평가에 쓸 수 있다"는 가능성만 보인 데 반해, 무엇이 신뢰할 수 있는 정책 평가자를 만드는지를 실로봇과 매칭된 32.4만+ 롤아웃으로 통제 비교한 첫 벤치마크(WMBench)와 설계 로드맵을 제시하고 이를 GigaWorld-1로 구현·전면 오픈소스화했다. 핵심 통찰은 평가자 품질이 단기 시각 리얼리즘이 아니라 장기·액션충실 롤아웃 일관성에 의해 좌우된다는 것이다.
강점
  • 대규모·통제 실험: 7개 비디오 월드모델 + 4개 액션 인코딩 + 32.4만+ 시뮬 롤아웃(실로봇 실행과 near 1:1 페어링) + 12,000+시간 학습 비디오 + CVPR 2026 GigaBrain 챌린지 커뮤니티 제출까지 포함 — 근거 폭이 넓다.
  • 액션 충실도에서 큰 격차: GigaWorld-1-Plus의 Trajectory Accuracy 0.3561 / JEPA Similarity 0.9337로 최고 베이스라인(Cosmos-Predict2.5 0.1770 / 0.6781)의 약 2배. 반면 Image Quality(0.6765)는 베이스라인과 비슷 — '화질보다 액션정합이 평가자 품질을 좌우'라는 주장이 표로 뒷받침됨.
  • 설계 로드맵이 구체적·재현 가능: EE-pose/ray map 공간 액션 조건, 계층적 메모리(short/mid/long + relative RoPE)로 장기 롤아웃 위치 드리프트 억제, DMD2 few-step distillation으로 추론 가속.
  • 코드·사전학습 체크포인트·데이터셋·툴킷 전면 오픈소스 + 프로젝트 페이지 공개.
약점
  • 평가 범위 한정(저자 인정): WMBench 8개 태스크군이 모바일 매니퓰레이션·in-hand dexterous·safety-critical 자율을 아직 커버하지 못함.
  • 비디오 중심 월드모델에만 초점 — structured state-space나 하이브리드 3D 접근의 trade-off는 미검증(저자 인정).
  • 완전 자동 대리평가는 아직 아님: VLM 보조 성공 라벨링도 불확실 케이스에서는 human verification에 의존.
  • 실제 정책 성공률과의 상관(ρ, Eq.4로 정의) 수치가 메인 표에 명시적으로 제시되지 않아 '평가자 정합' 정량 근거가 프록시 지표(AVG/Trajectory Acc) 중심. 저자도 contact-sensitive 실패에 대한 비디오모델의 낙관 편향을 남은 과제로 언급.

비디오 파운데이션 모델을 로봇 action 데이터로 파인튜닝한 video-action model(VAM/WAM)이 compositional 일반화 능력을 잃는 현상(video-action generalization gap)을 다룬다. 입력은 현재 프레임+지시문, 출력은 action chunk이며, action head가 예측된 미래 비디오 롤아웃에 얼마나 의존하는지를 attention 비율로 재는 Temporal Ratio(TR)를 도입하고, TR이 task 국면에 따라 요동치는 성질을 이용한 학습불필요 inference-time adaptive guidance를 제안한다. LIBERO OOD compositional과 실제 bimanual 로봇 과제에서 OOD-ID 일반화 격차를 크게 줄였다(LIBERO Avg OOD 26.8→59.4%, 실제 55.0→83.3%).

기여설계공간 탐색에서 표준적 설계 선택만으로는 일반화에 대한 emergent 설명 패턴이 없음을 보인 뒤, action head가 미래 롤아웃 대비 현재 프레임에 두는 attention 비율인 Temporal Ratio(TR)가 모델의 compositional 일반화 능력을 예측함을 최초로 제시한다. 나아가 TR이 planning 국면엔 상승·정밀 조작 국면엔 하강하는 내재적 요동을 활용해, 미래 조건 신호를 필요한 순간에만 동적으로 증폭하는 학습불필요 inference-time adaptive guidance로 gap을 완화한다.
강점
  • TR로 구조적 진단을 제공: action head의 future-latent 의존도(attention 비율)가 compositional 일반화 능력의 예측 지표임을 보여, 기존 설계공간에서 없던 emergent 패턴을 사후 설명함
  • 학습 불필요 inference-time guidance만으로 LIBERO Avg OOD를 π0.5 26.8%→59.4%, 실제 bimanual YAM 55.0%→83.3%로 대폭 향상하면서 ID 성능(≈94%)은 유지
  • TR이 task phase에 따라 자연히 요동(planning 상승, 정밀 조작 하강)함을 관찰하고 이를 guidance 가중치 s_i=clip(TR_i/TR_0-1,0,s_max)로 연결한 메커니즘 기반 설계

약점

LTC(Liquid Time-Constant) 동역학으로 belief를 갱신하고 AdaLN 변조로 디코더에 직접 주입한다. LIBERO 평균 96.85→98.75%, LIBERO-plus 91.4→93.77%, 메모리 진단 MIKASA ShellGameTouch 75.0%를 달성했다.

기여과거 관측을 단순한 passive history context로만 쓰던 기존 memory-VLA와 달리, TFP는 입력-의존 시간상수 τ를 가진 LTC 게이트(k_t=exp(−Δt/τ))로 접촉·해제·subgoal 전환 근처에서 belief를 능동 갱신하고, 이를 cross-attention이 아닌 AdaLN-style 변조로 flow-matching action decoder에 직접 주입해 누적된 시간 컨텍스트가 생성되는 action chunk 자체를 형성하게 한다.
강점
  • LIBERO 4-suite 평균 96.85→98.75%, 특히 장기 태스크 Long-10이 92.4→97.0%로 크게 향상, LIBERO-plus(Noise/Light/Background 교란)도 91.4→93.77%로 강건성 개선.
  • 실물 Galaxea A1에서 object swap 3/20→15/20, counting pick-place 8/20→18/20으로 π0.5 대비 큰 폭 향상, 메모리 진단 벤치 MIKASA ShellGameTouch 75.0% 달성.
  • LTC+AdaLN이 경량이라 추론 오버헤드가 거의 없고, 3.3B 모델로 7B급 baseline(OpenVLA-OFT 7.3B, MemoryVLA 7.3B, AVA-VLA 7.3B) 대비 우위를 보고.
  • Mechanistic 분석으로 write-gain이 manipulation event 근처에서 non-event 대비 약 6× 크고, hidden-state intervention으로 belief가 생성 action chunk를 인과적으로 변조함을 검증.
약점

입력은 라벨 없는 순수 human 비디오(로봇 액션·human측 주석 없음)와 frozen World-Action Model(LDA 백본), 출력은 그 human 시연 방향으로 조종된 로봇 액션이다. WAM-TTT는 human 비디오를 "모방할 궤적"이 아니라 frozen WAM 내부의 경량 adaptive memory(TTT fast-weight)에 self-supervised 비디오 예측으로 흡수해, 로봇 데모나 태스크별 파인튜닝 없이 로봇 파운데이션 모델을 새 태스크 변형·선호 행동으로 조종한다. 9개 실물 조작 태스크에서 in-context human-video conditioning baseline을 일관되게 능가한다.

약점

Initiation Safety: A Missing Dimension in Generalist-Robot Safety

필독 E 15완성도 45🎓 RSS 2026 Workshop on Rethinking Safety for Generalist Robots

제너럴리스트 로봇이 인사·초대받지 않은 grasp·개인공간 진입처럼 '되돌리기 어려운 첫 사회적 행동'을 애초에 시작해도 되는지를 판단하는 '개시 인가(initiation authorization)'를 별도의 안전 계층으로 제안한다. 입력은 시선·발화·접근·체류를 융합한 참여도 점수 $e_t$와 단계별 probe 규칙, 장소 대담성 설정 $\rho$이고, 출력은 $e_t \ge \tau(\rho)$이면서 probe 레벨 $\ge 2$일 때만 첫 발화를 허용하는 인가 게이트(및 감사용 여유 $\Delta_{init}=e_t-\tau(\rho)$)다. PAL Robotics ARI 휴머노이드 출입문 시나리오에 PAS(probe–authorize–speak)를 시범 구현하고 로그 1건으로 direct-init과 대비했으나, 정량 사용자 연구는 설계만 제시한 포지션/프레임워크 논문이다.

기여모션 안전·대화(내용) 안전과 구분되는 제3의 안전 계층으로 '개시 인가'를 명시적으로 분리·정의하고, 높은 참여도 점수나 자신감 있는 VLA rollout을 곧 행동 허가로 취급하는 기존 스택(post-plan VLA guardrail)과 대조한다. 되돌릴 수 있는 소규모 probe → 명시적 대담/보수 설정 → foundation-model 대화 직전 런타임 체크로 이어지는 PAS 절차와, 감사 가능한 개시 로그 스키마를 제안한다.
강점
  • 모션/대화 안전과 별개로 '말을 걸어도 되는가(사람을 봤다 ≠ 동의)'라는 과소 다뤄진 안전 차원을 분리해 문제화.
  • 구체적 메커니즘 제시: 참여도 $e_t$ + 4단계 probe + 인가 게이트($e_t\ge\tau(\rho)$ AND probe$\ge2$), 첫 발화 시점의 여유 $\Delta_{init}=e_t-\tau(\rho)$를 로그로 남겨 감사 가능하게 설계.
  • 실제 휴머노이드(PAL Robotics ARI) 출입문 시나리오에 PAS를 시범 구현.
  • 위조 가능한 3조건 피험자간 연구(PAS / direct-init / passive-wait)와 개방형 개시 로그 스키마를 함께 제안.
약점
  • 정량 결과 없음: 3조건 사용자 연구는 미래형으로 '설계'만 제시, 실제로는 로그 1건($\Delta_{init}$ 0.03 vs 0.06)의 예시(illustrative) 비교뿐.

VLA-JEPA

필독 B 70완성도 77✦📄 arXiv
2602.10098 · arXiv · code

멀티뷰 RGB 장면 카메라와 언어 지시를 입력받아, 장면이 어떻게 변할지를 픽셀이 아닌 추상 latent(JEPA embedding) 공간에서 상상하면서 연속 end-effector action chunk를 출력하는 VLA 정책이다. VLM 백본(Qwen3-VL-2B)이 latent action token과 action token을 내면, 작은 predictor(latent world model)가 미래 latent state를 예측하고 flow-matching head(DiT-B)가 행동을 생성한다. leakage-free state prediction으로 appearance bias·카메라 motion·정보 누출에 강건한 조작 정책을 달성했다(LIBERO 97.2%, LIBERO-Plus 79.5%).

기여기존 latent-action VLA pretraining은 latent가 raw 픽셀 변화에 anchoring돼 appearance bias·nuisance motion·information leakage에 취약했는데, VLA-JEPA는 미래 frame을 V-JEPA2 target encoder로 인코딩해 지도 target으로만 쓰고 학습자는 현재 관측만 보게 하는 leakage-free state prediction으로 이를 설계 차원에서 회피한다. 다단계 latent-action 파이프라인 대신 JEPA pretraining → action-head fine-tuning의 단순한 2단계 레시피로 일반화·강건성을 끌어올린 점이 새롭다.
강점
  • leakage-free state prediction: 미래 frame을 지도 target으로만 사용하고 학습자는 현재만 보게 해 latent action이 미래를 베껴 치팅하는 information leakage를 구조적으로 차단
  • 픽셀이 아닌 JEPA latent 공간에서 dynamics를 예측해 카메라 motion·배경 변화 등 action-irrelevant한 디테일에 강건
  • human video(alignment loss만) + robot data(alignment+action joint)의 cross-domain co-training으로 행동 라벨 없는 대규모 영상도 world-model 학습에 활용
  • LIBERO 97.2%, LIBERO-Plus 79.5%, SimplerEnv(Google 65.2%/WidowX 57.3%)로 π0.5·OpenVLA-OFT·UniVLA·villa-X 등 강한 baseline 대비 일관된 우위 + 코드·가중치 공개
약점
  • venue 없는 preprint(2026-02 공개) — peer review 미통과
  • 분류 경계가 모호: 미래 latent 예측은 pretraining 보조 목적이고 행동은 별도 flow-matching head라 ② WAM이되 ①(VLM+ActionExpert)로 읽을 여지도 있음(③/④는 분명히 아님)
  • action head가 predictor의 예측 future latent를 입력으로 받아 디코딩하는 구조가 아니라 단지 alignment loss target일 뿐 — '상상한 미래로 행동을 만든다'는 직관과는 다름
  • fine-tuning 시 proprioception 입력 사용 여부 미확인; arXiv abstract에는 수치가 없고 점수는 HTML 본문 표에서만 확인됨

DreamDojo

현재 카메라 view + 로봇 action chunk(4개 relative joint pose, +선택 텍스트)를 입력하면 다음 미래 video frame(픽셀)을 autoregressive하게 "꿈꾸는" action-conditioned 신경 시뮬레이터다. 라벨 없는 약 44k시간 egocentric human video로 사전학습한 generalist 로봇 world model로, downstream에서 가상 teleoperation·안전한 정책 평가·model-based planning에 쓰인다. 행동을 출력하지 않고 행동을 조건으로 받아 미래 관측만 생성하는 순수 WM이다.

기여선행 action-conditioned WM은 (영상, 행동) 쌍 라벨이 필요했지만 DreamDojo는 ~32-d KL 병목의 latent action model(Genie/LAPA 계열)로 라벨 없는 44k시간 human video에 pseudo-action을 제조해, generalist 로봇 WM을 human video만으로 대규모 사전학습한다. 또 Self-Forcing distillation으로 35-step bidirectional teacher를 4-step autoregressive student로 줄여 near real-time(10.81 FPS) 스트리밍 시뮬레이션("Simulation 2.0")을 실현했다.
강점

  • Self-Forcing distillation(35→4 step)으로 2.72→10.81 FPS near real-time 스트리밍 달성
  • sim 성공률이 real 성공률과 r=0.995로 강하게 상관 — 정책 평가용 시뮬레이터로서 신뢰도 높음

약점

강점

Learning Latent Action World Models In The Wild (JEPA-Wild)

필독 C 55완성도 52🔴 Meta (FAIR)📄 arXiv
2601.05230 · arXiv

행동 라벨이 전혀 없는 in-the-wild 인터넷 영상(YouTube-Temporal-1B, 약 20M 영상·10억 프레임)을 입력으로, V-JEPA 2 encoder가 만든 latent 공간에서 다음 frame latent를 예측하는 world model과, 연속 frame 사이의 latent action을 복원하는 IDM을 함께 학습한다. 출력은 예측된 next-frame latent이며, 작은 controller가 실 로봇 행동을 latent action으로 매핑해 현재 이미지+목표 이미지로부터 goal-conditioned 행동 sequence를 plan한다. 라벨 없이 학습했음에도 action-conditioned baseline과 유사한 planning 성능을 달성한다.

기여기존 latent-action world model이 로봇 시뮬·게임·조작 데이터에 국한되고 대개 vector quantization(VQ)으로 행동을 discretize했던 것과 달리, 풍부한 실세계 dynamics를 잡기 위해 VQ 대신 sparsity/noise 정보 병목으로 정규화한 연속(continuous-but-constrained) latent action 공간을 라벨 없이 IDM으로 복원한다. 이를 10억 프레임 규모의 uncurated 인터넷 영상으로 scale하고, 영상 간 환경 변화 transfer와 카메라 기준으로 localize된 행동 학습을 보이며, controller(universal interface)로 실 로봇 planning까지 연결한 점이 새롭다.
강점
  • 라벨 없는 uncurated 인터넷 영상(약 10억 프레임)으로 latent-action world model을 인터넷 규모로 scale
  • VQ(고정 codebook snap)가 무너뜨리는 풍부한 in-the-wild motion을, 연속이되 sparsity/noise 제약을 둔 latent action으로 포착 — VQ 대비 우위를 명시적으로 보임
  • 영상 간 환경 변화 transfer(예: 방에 사람이 들어오는 변화) 등 in-the-wild 특화 행동 학습을 입증
  • 가벼운 controller가 실 로봇 행동↔latent action을 매핑하는 universal interface 역할로, 라벨 없이 학습했음에도 action-conditioned baseline과 유사한 planning 성능 달성
약점
  • 공통 embodiment이 없어 학습된 latent action이 카메라 기준 공간에 localize되는 데 그침 — 일반화된 행동 의미를 직접 학습하지는 못함(저자 자인 limitation)
  • 정확한 planner(CEM/receding-horizon MPC)는 선행작 V-JEPA 2-AC에서의 추론이며 본 논문이 직접 명시한 것이 아님
  • 공식 코드·가중치 repo 미확인, 학습데이터(YT-Temporal-1B)도 GCS 버킷 셧다운으로 원본 프레임 접근 불가(메타데이터만, 유튜브 직접 수집 필요) — 재현성 제약
  • abstract·메타데이터에 정량 벤치마크 수치가 제시되지 않아 planning 성능이 'baseline과 유사' 수준의 정성 서술에 머무름; pixel-decoder loss가 정확히 L1인지 등 일부 세부는 미확인

InternVLA-A1

필독 D 35완성도 75📄 arXiv
2601.02456 · arXiv · code

멀티뷰 장면 이미지(현재+최근 과거 frame) + 언어 지시(+로봇 proprioception q_t)를 입력받아, 하나의 Mixture-of-Transformers 안에서 미래 visual latent(foresight)를 상상하고 그 상상에 조건화된 다음 action chunk a_{t:t+k}를 함께 출력하는 범용 로봇 조작 정책이다. understanding·generation·action 3개 expert가 단일 masked self-attention을 공유해 의미 이해와 미래 상상이 동시에 행동을 안내한다. pi0.5 대비 dynamic 조작에서 +26.7%를 보고했다.

기여표준 MLLM 기반 VLA는 물리 모델이 없어 dynamic task에 약하고, 순수 영상예측 WM 정책은 시각적으로 brittle하고 의미 grounding이 약했다. InternVLA-A1은 의미 이해(understanding expert)와 미래 visual foresight 생성(generation expert), 행동 생성(action expert)을 별도 모듈 직렬 연결이 아니라 단일 unified masked self-attention을 공유하는 하나의 Mixture-of-Transformers로 융합하고, blockwise mask로 understand→imagine→act 일방향 흐름을 강제해 상상한 미래가 항상 행동을 조건화하도록 만든 점이 새롭다.
강점
  • 미래 상상(visual foresight)과 행동을 하나의 self-attention을 공유하는 통합 network에서 co-train해, 별도 영상생성기→정책 파이프라인의 누적 오류 없이 의미 추론과 미래 예측이 함께 행동을 안내한다
  • blockwise mask로 understand→imagine→act 일방향 흐름을 강제해 행동이 항상 foresight에 종속되게 구조화 — 상상 영상 오류가 행동을 직접 망치는 것을 완화
  • 실로봇+시뮬+egocentric human video 혼합 692M frame으로 사전학습해 sim-to-real gap을 줄이고, dynamic manipulation에서 pi0.5 대비 +26.7%로 표준 VLA의 약점인 동적 task를 크게 개선
  • code(github.com/InternRobotics/InternVLA-A1)와 3B 가중치, InternData-A1 데이터셋, RoboTwin 2.0 벤치마크가 (부분) 공개되어 재현/확장 가능
약점
  • 벤치마크 수치(+4.4% static · +2.6% RoboTwin 2.0 · +26.7% dynamic)는 모두 저자 보고이며 독립 재현이 아님
  • 예측 미래에서 행동을 역산하는 별도 IDM이 없고 foresight가 동등한 target이 아니라 보조 신호여서, ④ action-centered WAM 판단은 방어 가능한 해석일 뿐 — 미래 생성과 행동을 진짜 co-train하므로 ② WAM으로 볼 여지도 있음(논문은 이 4-패러다임 분류를 정의하지 않음)
  • 692M frame 전체 혼합(raw 실로봇+human video 포함)이 단일 릴리스로 공개되는지는 미확인이고, 2B 모델 가중치는 미공개(3B만 공개)
  • Cosmos VAE tokenizer는 frozen이고 generation은 픽셀 복원이 아닌 저차원 latent forecast라, 상상 품질이 tokenizer의 표현력에 묶이며 정밀한 시각적 미래가 필요한 task에서의 한계는 미평가

기여기존 정책(OpenVLA, Octo)이 행동을 discrete token으로 잘라 정밀 고주파 동작에 거칠고 학습 장면 근처에서만 동작했던 한계를, 사전학습된 VLM은 그대로 유지하고 별도 flow-matching action expert를 붙여 noise→연속 action을 흘려보내는 방식으로 해결했다. 두 expert를 shared self-attention(block-wise causal mask)으로만 연결해 VLM 지식을 보존하면서 정밀 연속 제어를 얻고, π0.5는 "먼저 말로 생각(subtask) → 행동" co-training으로 미지 환경 일반화까지 확장한 점이 새롭다.
강점
  • discrete token 대신 flow matching으로 H=50, ~50 Hz의 정밀·연속·multimodal action chunk를 생성 — 빨래 개기 같은 dexterous 고주파 동작 가능

SuSIE (Zero-Shot Robotic Manipulation with Image-Editing Diffusion)

필독 D 35완성도 56📄 arXiv
2310.10639 · arXiv · code

로봇 카메라 RGB 한 장 + 텍스트 명령("냄비에 숟가락 넣기")을 입력하면, 사전학습 image-editing diffusion(InstructPix2Pix)이 ~20 step 뒤 장면이 어때야 하는지를 사실적인 subgoal RGB 이미지로 그려내고, 별도의 작은 goal-conditioned diffusion BC 정책(gc_ddpm_bc)이 현재 이미지와 그 subgoal 이미지를 받아 action chunk(horizon=4)를 출력한다. 이 imagine-then-act 루프를 연쇄해 긴 task를 짧은 시각 목표들로 분해·완수하며, CALVIN(zero-shot ABC→D) SOTA와 실세계 Bridge-style task에서 RT-2-X 능가를 달성했다.

기여미래-상태 추론을 인터넷 사전학습 image generator(InstructPix2Pix)에 떠넘겨, 시스템이 넓은 시각/의미 일반화를 물려받아 새 객체·지시를 zero-shot으로 처리하면서도 low-level 정책은 단순·language-agnostic하게 유지한 것이 선행 대비 새로움이다. 예측 대상이 latent code가 아니라 눈에 보이는 RGB subgoal 이미지이고, 따로 학습된 WM(diffusion subgoal generator)과 IDM 역할 정책을 그 "이미지 한 장"으로만 연결하는 hierarchical 분해를 제시했다.
